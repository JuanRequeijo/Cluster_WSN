Este projeto (wsnee5) contempla a última versão estável do código da simulação do nosso algoritmo (WsnEE - Wireless Sensor Networks Energy Efficient) que utiliza apenas um 
tipo de dados (p. ex.: temperatura, umidade, luminosidade etc) para fazer a predição de dados.
OBS.: Esta versão foi a utilizada para os testes do trabalho enviado para o congresso ISCC2014.
